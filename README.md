# Expense Manager Frontend

## 📌 Project Description

Expense Manager Frontend is a responsive web application built to interact with the Expense Manager Backend APIs.  
It allows users to manage daily expenses through a clean and user-friendly interface.

This project demonstrates frontend development skills including API integration, state management, form validation, and responsive UI design.

---

## 🚀 Tech Stack

- React.js (or Angular / Vue – update if needed)
- JavaScript (ES6+)
- HTML5
- CSS3 / Bootstrap / Tailwind (update if needed)
- Axios / Fetch API
- REST API Integration

---

## ✨ Features

- Add new expense
- Update existing expense
- Delete expense
- View all expenses
- Filter expenses by category
- Filter expenses by date
- Form validation
- Responsive design
- API integration with backend
